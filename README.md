NutriMind: Nutrition and Energy Tracking Database 

About the Project: NutriMind is a database project that helps users track their food intake, water consumption, and physical activities. The main goal of this project is to design and implement a structured relational database using SQL, Python, Html, CSS. This project was developed as part of a database systems course.

Features: Track daily food and calorie intake,Track daily water intake (HydraTrack), Track exercise time and exercise types, Calculate weekly calorie consumption, Provide food allergen alerts, Store and manage data in a structured database, User can listen to music according to their mood, Track BMI

Database Design: The database follows basic relational database principles. Main tables include: Users Food Nutrition Records Hydration Records Exercise Records Allergens Music BMI Admin The design uses primary keys, foreign keys, normalization, and SQL constraints to ensure data integrity.

Tools and Technologies: Database: MySQL / PostgreSQL

Query Language: SQL 

Design Tool: ER Diagram Project Structure, Frontend: HTML, CSS, Javascript, Backend: Python

How to Run the Project: Clone or download the repository. Open your database management system. Run the SQL files to create tables and insert data. Execute sample queries to test the database. Sample Queries, Daily and weekly calorie calculation, BMI Calculation ,Add Music, Daily water intake summary ,Exercise activity summary, Foods containing allergens

Purpose: This project is intended to help students understand relational database design and practical SQL usage through a real-world example. Future Improvements Add a web or mobile interface Add data visualization Improve nutrition analysis
